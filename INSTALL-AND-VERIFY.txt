INSTALLATION AND VERIFICATION

1. Extract the ZIP.
2. Open a terminal in the xai-intelligence-workspace folder.
3. Run: npm install
4. Run: npm run dev
5. Open: http://localhost:3000
6. Before submission, run:
   npm run typecheck
   npm run lint
   npm run format:check
   npm run build
   npm run test:e2e

The package installation could not be completed inside the file-generation environment because the npm install command exceeded the available execution timeout.
